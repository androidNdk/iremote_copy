Related Projects

CommonKit

iRemote and RemotePC data model, concurrency schema and object oriented network interaction are based on CommonKit project.

http://commonkit.googlecode.com/

hielog

hielog (Hierarchical Logging) is lightweight logging solution for C++. This library is utilized for application on device testing.

http://hielog.googlecode.com/

boost::asio

Low level network interaction in CommonKit is based on cross-platform wrapper over BSD sockets provided by boost::asio.

http://boost.org/
